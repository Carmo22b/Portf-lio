## Como rodar o projeto

Clone o repositório:

```bash
git clone https://github.com/Carmo22b/Portf-lio.git
```

Acesse a pasta do projeto:

```bash
cd Portf-lio
```

Instale as dependências:

```bash
npm install
```

Inicie o projeto:

```bash
npm start
```
